Packet Sniffer
==============

A simple packet sniffer made using Python3.

You can sniff packets based on:
* protocol
  - HTTP
  - FTP
  - TELNET
  - TCP
  - UDP
* Post number
* IP address

Usage
```python
sudo python3 mysniffer.py <filter> <filter> <filter>
```

You can apply no filters or at max three filters only.
